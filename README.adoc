= Ansible with Vagrant

WARNING: `controller` can't ping `node1` nor `node2`

== Operation

=== Vagrant

----
vagrant up
vagrant ssh
----

.https://www.vagrantup.com/intro/getting-started/teardown.html
----
vagrant suspend
vagrant halt
vagrant destroy
----

[WARNING]
====
vagrant destroy --force
====

=== Ansible

----
ansible all -m ping -i inventory.yml
----

== References
